# Figma Design Challenge — "Design is code"

A keynote-style talk deck for the Figma Design Challenge 2026, by YJ Soon (Tinkercademy).

## Contents

- **`index.html`** — the live deck. Open in any modern browser. Click, `→`, or space to advance; each click reveals the next build. Every state is deep-linkable at `#phase1` … `#phase21`.
- **`figma-design-challenge.pptx`** — high-fidelity PowerPoint export (16:9, one slide per click-state, fade transitions on click). Slides are rendered images, so layout and custom fonts match the web deck exactly — edit `index.html` for any changes, then re-export.
- **`speech.md`** — the speaker script, with slide markers and `→` reveal cues (~8–9 min).
- **`pdf-slides/`** — static image exports of the slides.

## Presenting

Open `index.html` full-screen and present from the browser, or open the `.pptx` in PowerPoint / Keynote.
